# Student Result Management & Performance Analytics (MySQL)

This project manages student academic records and performs detailed
performance analysis using MySQL. It includes student details,
subjects, marks, grades, ranking, and performance reports.

The project simulates a real-world college/university result system.

## Technologies Used
- MySQL 8.0
- Relational Database Concepts

## SQL Concepts Covered
- DDL & DML
- Primary & Foreign Keys
- Joins
- Aggregate Functions
- CASE Statements
- Subqueries
- Views
- Window Functions (RANK)
- Date Functions

## Use Case
- College Result Processing
- Student Performance Evaluation
- Academic Analytics
